## Project Setup

### 1. Clone Git Repository
`Clone Git Repository` -> `https://github.com/solm0/kmuvcd.git` -> `Clone from URL` -> 파일 위치 선택 -> `Terminal>New Terminal`

### 2. Install dependencies
```
npm install
```

### 3. Run the project
```
npm run dev
```
최초 로딩 1분

### 4. Commit
수정 후 `Source Control`에서 `Commit`
